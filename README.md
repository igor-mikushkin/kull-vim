# KULL  

- Modified fork of skull-vim
- Mostly monochrome-ish and green theme

## Installation

Install manually or use a package manager:

```viml
" vim-plug
Plug 'igor-mikushkin/kull-vim'
" NeoBundle
NeoBundle 'igor-mikushkin/kull-vim'
" Vundle
Plugin 'igor-mikushkin/kull-vim'
```

```viml
colorscheme kull
```
